# ASMR Sound Assets

Place MP3 files here with the following names:

| File                   | Emotion    | Description          |
|------------------------|------------|----------------------|
| pencil_bright.mp3      | joy, hope  | Bright pencil sound  |
| pencil_slow.mp3        | sadness    | Slow pencil sound    |
| pencil_sharp.mp3       | anger      | Sharp pencil sound   |
| pencil_trembling.mp3   | fear       | Trembling pencil     |
| pencil_erasing.mp3     | regret     | Erasing sound        |
| pencil_gentle.mp3      | calm, love | Gentle pencil sound  |
| pencil_fast.mp3        | excitement | Fast pencil sound    |
| pencil_default.mp3     | fallback   | Default pencil sound |

These will be accessible at `/sounds/<filename>` in the browser.
